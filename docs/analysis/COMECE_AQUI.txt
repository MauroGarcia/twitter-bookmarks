🎓 GLOSSÁRIO RÁPIDO
═══════════════════════════════════════════════════════════════════════════════

FTS5            = Full-Text Search do SQLite (busca em texto)
IPC             = Inter-Process Communication (Electron)
Transaction     = Grupo de SQL que falha junto ou sucede junto
Cascata         = Delete que propaga (ON DELETE CASCADE)
E2E             = End-to-End (teste de usuário completo)
Smoke Test      = Teste rápido e crítico
Virtualization  = Renderizar apenas elementos visíveis (performance)
Debounce        = Delay antes de executar (evita spam)
