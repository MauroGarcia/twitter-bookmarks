│ T=50ms:   GET /bookmark/1/tags               ✓ 20ms          │
│ T=70ms:   GET /bookmark/2/tags               ✓ 20ms          │
│ T=90ms:   GET /bookmark/3/tags               ✓ 20ms          │
│ ...                                                           │
│ T=2050ms: GET /bookmark/100/tags             ✓ 20ms          │
│                                                               │
│ T=2070ms: UI carrega com 100 bookmarks       (TOTAL: 2070ms) │
│                                                               │
│  ❌ 101 API calls (1 bookmarks + 100 tags)                    │
│  ❌ Execução SEQUENCIAL (não paralela)                        │
│  ❌ 2+ segundos de lag perceptível                            │
│  ❌ Com 1000 bookmarks: ~20 segundos!                         │
│                                                               │
└──────────────────────────────────────────────────────────────┘

DEPOIS - Network Timeline (Backend otimizado):
┌──────────────────────────────────────────────────────────────┐
│ T=0ms:    GET /bookmarks                     ✓ 50ms          │
│           → [bookmark1, bookmark2, ..., bookmark100]         │
│                                                               │
│ T=50ms:   POST /bookmarksTags                ✓ 120ms         │
│           Request: {bookmarkIds: [1, 2, ..., 100]}           │
│           Response: {                                        │
│             "1": [{id:1, name:"React", ...}],               │
│             "2": [{id:2, name:"Vue", ...}],                 │
│             ...                                              │
│             "100": [{id:100, name:"TypeScript", ...}]       │
│           }                                                  │
│                                                               │
│ T=170ms:  UI carrega com 100 bookmarks + tags  (TOTAL: 170ms)│
│                                                               │
│  ✓ 2 API calls só (1 bookmarks + 1 tags batch)               │
│  ✓ Execução paralela possível                                │
│  ✓ 170ms vs 2070ms = 12.2x mais rápido!                      │
│  ✓ Com 1000 bookmarks: <300ms (escalável)                    │
│                                                               │
└──────────────────────────────────────────────────────────────┘
